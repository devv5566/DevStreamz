# WatchFlix by DevStreamz

WatchFlix is a Stremio streaming addon powered by DevStreamz.

## Included Providers

- ShowBox
- 4KHDHub

## Quick Start

1. Install dependencies:
   - `npm install`
2. Start server:
   - `npm start`
3. Open setup page:
   - `http://localhost:7777`
4. Generate manifest and install in Stremio.

## Notes

- ShowBox may require a valid FebBox cookie for best results.
- 4KHDHub works without ShowBox cookie.

## Project Identity

- Addon name in Stremio: `WatchFlix | DevStreamz`
- UI branding: `WatchFlix by DevStreamz`
