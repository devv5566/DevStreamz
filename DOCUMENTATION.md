# WatchFlix Deployment Guide

This guide is for deploying `WatchFlix by DevStreamz`.

## Requirements

- Node.js 16+
- npm

## Install

```bash
npm install
```

## Run

```bash
npm start
```

Server starts on `http://localhost:7777`.

## Configure

Open:

- `http://localhost:7777`

Use the setup page to:

- Set optional ShowBox cookie
- Set optional region
- Enable/disable ShowBox and 4KHDHub
- Generate install URL for Stremio

## Branding

- Manifest name: `WatchFlix | DevStreamz`
- Web setup title: `WatchFlix Setup`
